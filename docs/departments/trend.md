# Trend Department

트렌드 디파트먼트는 육아 트렌드, 지역 부모 커뮤니티 흐름, 인기 행사, 시즌 이슈를 다룬다.

- `k-parent-festival-finder`

## Shared patterns

- 트렌드성 정보는 출처, 날짜, 지역, 표본 편향을 함께 표시한다.
- 후기와 공식 정보를 섞지 않는다.
- 추천은 현재성 확인이 중요하므로 최근 출처를 우선한다.
- 실제 크롤링 결과는 source URL, fetched_at, freshness, confidence를 함께 저장한다.
- 유튜브/콘텐츠 추천은 조회수만 보지 않고 채널 신뢰도, 연령 적합성, 광고성, 교과 연결성을 함께 본다.
