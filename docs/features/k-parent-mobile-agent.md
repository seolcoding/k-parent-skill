# 모바일 에이전트 플레이스홀더

`k-parent-mobile-agent`는 일반 부모가 모바일에서 접속해 자녀 일정, 학교/학원 정보, 지역 추천, 리마인더를 에이전틱하게 사용하는 서버 배포형 제품 표면이다.

## 현재 범위

- 보호자 계정과 자녀 프로필
- 학교/학원/지역/관심사 기반 모바일 홈
- 에이전틱 명령 입력
- 캘린더/리마인더 후보 확인
- 데이터 출처와 갱신 시각 표시

- 서버 런타임과 사용자 세션
