# Research Notes

This directory records product and architecture research that informs `k-parent-skill`.

Research notes are not executable skill instructions. Promote durable decisions into:

- `docs/architecture.md`
- `docs/productization-gstack.md`
- `docs/roadmap.md`
- `docs/departments/*.md`
- relevant `docs/features/*.md`

When a research note includes time-sensitive data, verify the original source before turning it into runtime behavior.
