# NEIS 알레르기 코드 (1~19)

학교 급식(NEIS `mealServiceDietInfo`)의 메뉴 뒤 괄호 숫자는 알레르기 유발식품 번호다.
단일 출처는 `packages/k-parent-source-neis/src/parse.js`의 `ALLERGEN_LABELS`이며, 이 표는 그 사본이다.
표시할 때는 번호를 아래 라벨로 바꿔 부모가 바로 이해하게 한다.

| 번호 | 식품 |
| --- | --- |
| 1 | 난류 |
| 2 | 우유 |
| 3 | 메밀 |
| 4 | 땅콩 |
| 5 | 대두 |
| 6 | 밀 |
| 7 | 고등어 |
| 8 | 게 |
| 9 | 새우 |
| 10 | 돼지고기 |
| 11 | 복숭아 |
| 12 | 토마토 |
| 13 | 아황산류 |
| 14 | 호두 |
| 15 | 닭고기 |
| 16 | 쇠고기 |
| 17 | 오징어 |
| 18 | 조개류(굴, 전복, 홍합 포함) |
| 19 | 잣 |

예: `돈육김치찌개(5.9.10)` → 대두·새우·돼지고기 포함.
